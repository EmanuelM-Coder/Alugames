# 🎮 Alugames

Projeto desenvolvido junto à **Alura**, simulando uma **locadora de jogos online**.  
Com o **Alugames**, você pode facilmente **alugar** e **devolver** seus jogos favoritos de forma simples e intuitiva.

---

## 🚀 Tecnologias utilizadas

- **HTML**
- **CSS**
- **JavaScript**

---

## 🌐 Acesse o projeto

👉 [Clique aqui para acessar o Alugames no Vercel](https://alugames-ruddy.vercel.app/#)

---

## 📖 Como funciona

1. Escolha um jogo disponível.
2. Clique em **Alugar**.
3. Quando terminar de jogar, clique em **Devolver**.

Simples assim! 🎉
